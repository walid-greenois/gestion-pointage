# Video Background Files

Place your login background video here as `login-bg.mp4`.

You can use any MP4 video file. The recommended specifications are:
- Format: MP4 (H.264 codec)
- Resolution: 1920x1080 (Full HD)
- Duration: 5-10 seconds
- Size: Keep under 10MB for optimal loading

The video will be set to autoplay, muted, and loop continuously in the background.

### Example sources:
- Pexels Videos: https://www.pexels.com/search/videos/
- Pixabay Videos: https://pixabay.com/videos/
- Unsplash Videos: https://unsplash.com/videos

You can rename any video to `login-bg.mp4` and place it in this directory.
